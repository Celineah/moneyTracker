## ADDED Requirements

### Requirement: Multi-Screen Tab Navigation
The system SHALL provide a bottom tab navigation interface to switch between the main expense dashboard and the statistics view.

#### Scenario: Switching to Statistics
- **WHEN** the user taps the "Statistics" tab in the bottom bar
- **THEN** the system MUST navigate to the Statistics Screen.

#### Scenario: Switching to Home
- **WHEN** the user taps the "Home" tab in the bottom bar
- **THEN** the system MUST navigate to the Home Screen.
